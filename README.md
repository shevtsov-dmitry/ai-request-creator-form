# ai-request-creator-form

**Description:** 

The "AI request creator form" is a simple web/desktop application that makes it easy to create and manage requests for AI. You can quickly type in your request and any important details, then copy the results as either JSON or TOON for use with other programs.


**Key Features:**

*   Lightweight multiplatform compiled app with tauri framework.
*   User Input: Easy text area for requests.
*   Context: Manage multiple context inputs.
*   Keyboard Shortcuts.


**Technologies:**

*   React
*   Bun
*   Tauri
